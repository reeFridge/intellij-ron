# intellij-ron

## About

RON plugin for IntelliJ IDEs

## Features

- Syntax highlighting

## Build

### Requirements

* Java 8

### Instructions

1. Clone project repo
```
git clone https://github.com/reeFridge/intellij-ron
```
2. Enter to the project dir
```
cd intellij-ron
```
3. Run build task
```
./gradlew buildPlugin
```

## Test

* Run auto-tests
```
./gradlew test
```
* Run IDE instance with installed plugin
```
./gradlew runIde
```
